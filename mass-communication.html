理论与范式

理论 = 解释现象的概念体系 + 命题 + 假说
科学范式演进：前科学 → 常规科学 → 危机 → 范式革命
三种范式：
1. 客观经验主义：社会有客观规律、可测量、价值中立、数学描述
2. 诠释经验主义：社会无规律只有规则；行为基于主观诠释；意义不可量化但可解释
3. 批判理论（法兰克福学派）：反对技术理性；揭露资本主义意识形态；强调阶级/性别/民族压迫；研究即政治实践

功能、影响与受众

知沟假说：议题热度下降 → 知沟缩小
创新扩散：弱连接注入活力；两级传播传统；大众媒介直通个人
认知基模：补充细节 + 情感倾向（如“慈祥老人”好感）
沉默的螺旋：少数群体更愿公开表达
议程设置：媒体不能决定你怎么想，但能决定你想什么
培养理论：电视塑造“媒介现实”；商业与叙事强化稳定讯息系统

文化研究、权力与意识形态

文化 = 社会存在与意识的协商（汤普森）
工人阶级青年“反文化”：光头、飞车、黑人音乐 → 象征性反抗 + 弥补社群衰落
阿尔都塞：意识形态通过“召唤”建构主体（“这是在说我”）
福柯：权力是网络，非压制而是生产性（生成话语/知识）；人人皆为权力执行者
通俗文化：微观政治中的意义斗争 ≠ 宏观行动
“游牧的主体性”：在多重身份间游移
学术场非自治：“叫兽”为权力代言，非为生民立命

媒介理论（麦克卢汉 & 伊尼斯）

印刷媒介 → “想象的共同体”（安德森）
伊尼斯：偏时间媒介（重传统/集权） vs 偏空间媒介（重扩张/离心）
麦克卢汉：
• 媒介即讯息
• 热媒介（广播）vs 冷媒介（电视）→ 参与度差异
• 电视是冷媒介 → 需互动（选秀、短信）
媒介建构世界：旅行 = 验证媒介符号
消费 = 消费品牌故事（符号价值 > 使用价值）
“伪后台”：直播私生活 = 包装后的前台
